
## Map application that displays point marks and regional lists

Run Project : 
1. cd bvt
2. npm start 
3. Project running at: http://localhost:3000

Run JSON-Server : 
1. cd json-server
2. json-server --watch db.json --port=3001 
3. JSON-Server running at: http://localhost:3001

Tech Stack:
- React
- Leaflet
- React Redux
- React Router
- Bootstrap
- Material-UI
- i18next
